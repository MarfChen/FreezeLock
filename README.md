# FreezeLock
一个开源的有效关闭自动更新的小工具
1、使用

compiled.7z为已经编译并打包好了的版本，可直接使用

请关闭杀毒软件后使用！

通常情况下一次禁止永久有效，所以非必要不勾选自启动。使用完后可删除此软件已节约空间。

程序包中有一个名为FreezeLockCmd.exe的可执行文件，该文件旨在配合主程序阻止相关设置被篡改，单独执行该文件同样能阻止自动更新。
